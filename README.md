# NovetCoin / NVT

NovetCoin (NVT / NOVET) is a Polygon-based utility token for the Novet creative posting platform.

Novet is a creative posting platform for novels, illustration-based works, and creator-driven digital content.  
NVT is designed to support creator rewards, paid creative works, purchasing Novet Points (NP), and wallet-based platform functions.

## Utility / Use Cases

NovetCoin / NVT is intended to be used for platform-related utility functions within the Novet ecosystem.

Examples of intended use cases include:

- Creator rewards based on Novet platform rules
- Paid novels, illustration-based works, and creator-driven digital content
- Purchasing Novet Points (NP)
- Wallet-based deposit and withdrawal functions
- Platform-related liquidity support and ecosystem operations

## Novet Points / NP

Novet Points (NP) are internal platform points used for service fees and platform features within Novet.

NVT and NP have different roles.  
NP is mainly used for internal platform services, while NVT is related to creator rewards, wallet functions, and Web3-related platform utility.
